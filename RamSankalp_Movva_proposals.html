A unified movie booking platform that aggregates showtimes from multiple Belgian theatres, letting users easily browse, book, and review movies in one place.

Problem & Target Audience

Moviegoers in Belgium often need to visit different cinema websites like Kinepolis and UGC to find showtimes and compare options. MovieMate simplifies this by displaying all theatre schedules on one platform. It‚Äôs designed for movie lovers who want a convenient, secure, and modern way to explore movies and book tickets online.

Core Features

  • User Registration & Login: Secure authentication and optional Google OAuth.
  • Profile Management: Update profile info, view booking history, and save favorite movies.
  • Movie Discovery: Fetch and display movies from TMDb API with posters, genres, ratings, and trailers.
  • Unified Showtimes: Display showtimes from multiple Belgian theatres in one interface.
  • Booking System: Select movie, theatre, and time; confirm booking with a simulated payment system.
  • Admin Dashboard: CRUD management of theatres, movies, showtimes, users, and bookings.
  • Map Integration: Show theatre locations on Google Maps.
  • Seat Selection: Interactive seat map with live seat availability.
  • Ratings & Reviews: Users can review movies and theatres after viewing.
  • Automated Emails: Booking confirmations and reminders via n8n or Zapier.
